The Story

This project started with a simple question: how exceptional was Sir Alex Ferguson's 27-year tenure at Manchester United in the context of modern football? This visualization attempts to answer that by mapping the tenures of managers at Europe's top clubs since the 1955/56 season, the dawn of European club competitions.

The heatmap visualizes managerial stability. Each row represents a club, and each colored block represents a season. The color scheme indicates the manager's tenure at that point: a fiery red for a one-season stint, transitioning through shades of green for longer, more stable periods. Trophies won in a season are marked with shield icons, allowing for a visual analysis of the correlation between stability and success.

This project is an ongoing collaboration between Jasper Koning and Google's Gemini, built iteratively with D3.js for the visualization and Google Firestore for data management.

Changelog

v3.0 - The Great Migration (June 18-19, 2025)

  • Recognized that the CSV workflow was becoming a bottleneck.
  • Completely re-architected the data backend, migrating from a static engeland.csv file to a scalable Google Firestore database.
  • Data was separated into two collections (coaches and seizoenen) for data integrity and easier maintenance.
  • Developed and executed a one-time migration script to transfer all existing data to the new structure.
  • Rewrote the script.js to fetch and join data from Firestore and successfully fixed all outstanding bugs related to the interactive tooltip.
  • Milestone: Project is now scalable and ready for Fase 3.

v2.0 - The Polish & Interactivity (June 18, 2025)

  • Completed Fase 1 (Visual Polish): redesigned the page layout, improved the X and Y axes (including club logos), and added a comprehensive legend.
  • Initiated Fase 2 (Interactivity) by adding the core tooltip functionality.
  • Encountered and debugged several critical issues related to data quality and buggy tooltip behavior, leading to the strategic decision to migrate the data backend.

v1.0 - The Foundation (June 17-18, 2025)

  • Established the project's core principles and data structure.
  • Built the initial D3.js heatmap visualization based on a CSV file.
  • Iteratively refined the visual logic, including the color scheme for manager tenure, the representation of trophies, and the merging of coach periods into single visual blocks.
  • Set up the project on GitHub and deployed the first live version via GitHub Pages.
